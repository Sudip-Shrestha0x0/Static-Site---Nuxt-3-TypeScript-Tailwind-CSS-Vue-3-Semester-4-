## Project Description

This is a personal hobby site showcasing my diverse interests and pursuits, built using Nuxt as a static site generator. The site features a dark, modern design with responsive layouts optimized for all screen sizes.

## Technologies Used

- Nuxt 3
- Vue 3
- Tailwind CSS
- TypeScript

## Setup Instructions

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/shre0032/mtm6407-static-site-shre0032.git
cd mtm6407-static-site-shre0032
```

2. Install dependencies:
```bash
npm install
```

3. Run development server:
```bash
npm run dev
```

4. Build for production:
```bash
npm run generate
```

## Deployment

The site is deployed on Netlify and can be accessed at the URL provided in the assignment submission.
